## [0.1.0] — 2026-07-01

### Added

- Next.js project with TypeScript and Tailwind CSS v4
- shadcn/ui with button, separator, tooltip components
- Feature-oriented folder structure
- Google Colab-inspired dark theme
- Sidebar navigation with collapsible support
- Top navbar with mobile menu toggle
- App shell composing sidebar + navbar + main content
- Dashboard landing page with welcome message and empty-state card
- Navigation constants as single source of truth
- Project documentation (ROADMAP, ARCHITECTURE, DESIGN_SYSTEM, CONTRIBUTING, CHANGELOG)
- README with project overview and setup instructions
